# 개인정보 처리방침

최종 업데이트: 2026-04-24

## 1. 개요

New2Signal("서비스")은 RSS 피드에서 뉴스를 수집하고, 기사를 처리 및 분류하며, 구조화된 시장 브리핑을 생성하여 디스코드 서버에 전달하는 디스코드 봇입니다.

본 개인정보 처리방침은 어떤 정보가 수집되는지, 해당 정보가 어떻게 사용되는지, 얼마나 오래 보관되는지, 그리고 사용자 또는 서버 관리자가 정보 수정 또는 삭제를 어떻게 요청할 수 있는지를 설명합니다.

## 2. 수집하는 정보

New2Signal은 서비스 운영에 필요한 범위 내에서 다음 정보를 수집 및 저장할 수 있습니다.

### 2.1 디스코드 서버 정보

* 디스코드 서버 ID
* 디스코드 서버 이름
* 서버별 봇 설정
* 설정된 브리핑 전달 채널 ID
* 브리핑 전달 시간
* 알림 활성화 또는 비활성화 상태
* 선택된 관심 카테고리
* 시간대 및 관련 서버 설정

### 2.2 디스코드 사용자 정보

* 디스코드 사용자 ID
* 사용자 이름
* 표시 이름
* 해당되는 경우 서버 내 닉네임

### 2.3 서버 멤버십 정보

* 봇이 사용되는 서버와 사용자 간의 관계 정보
* 서비스 운영 또는 감사 목적에 필요한 경우 가입 관련 정보 또는 마지막 확인 시점 관련 메타데이터

### 2.4 감사 및 사용 로그

* 사용자가 봇을 통해 수행한 작업
* 명령어, 버튼, 선택 메뉴, 모달 등을 통해 변경한 설정
* 작업 발생 시각
* 추적되는 설정 변경의 이전 값과 변경 후 값

### 2.5 뉴스 및 브리핑 데이터

* 브리핑 생성을 위해 필요한 RSS 기사 메타데이터 및 정규화된 기사 내용
* 분류된 기사 데이터
* 생성된 일일 브리핑 데이터
* 기사 수 및 브리핑 대상 기간 관련 메타데이터

## 3. 정보 이용 목적

수집된 정보는 서비스 제공 및 개선에 필요한 범위 내에서만 사용됩니다. 사용 목적은 다음과 같습니다.

* 설정된 디스코드 채널로 예약 브리핑 전달
* 슬래시 명령어를 통한 저장된 브리핑 표시
* 서버별 설정 저장
* 대화형 설정 기능 제공
* 주요 설정 변경 또는 상호작용 이벤트에 대한 감사 로그 유지
* 서비스 안정성 모니터링, 디버깅 및 남용 방지
* 분류 및 브리핑 품질 개선

## 4. 법적 및 기능적 근거

데이터는 봇의 명시된 기능을 운영하는 데 필요한 범위 내에서만 처리됩니다. New2Signal은 디스코드 봇 기능, 서버 설정, 로그 기록, 브리핑 전달에 필요한 정보를 초과하여 불필요한 개인정보를 수집하는 것을 목적으로 하지 않습니다.

## 5. 제3자 서비스

New2Signal은 서비스 운영을 위해 다음과 같은 제3자 서비스에 의존할 수 있습니다.

* 봇 플랫폼 기능을 제공하는 Discord
* 뉴스 수집을 위한 RSS 피드 제공자
* 호스팅 또는 인프라 제공자
* 데이터베이스 제공자
* 배포 설정에서 활성화된 경우 LLM 또는 AI API 제공자

LLM 제공자가 사용되는 경우, 기사 내용 또는 브리핑 생성 관련 입력값이 분류, 요약 또는 브리핑 생성을 위한 목적으로만 해당 제공자에게 전송될 수 있습니다.

New2Signal은 사용자 데이터를 판매하지 않습니다.

## 6. 데이터 보관

데이터는 서비스 운영, 안정성, 감사 또는 문제 해결에 합리적으로 필요한 기간 동안만 보관됩니다.

일반적인 보관 항목은 다음과 같습니다.

* 서버 설정: 봇이 설치되어 있거나 설정이 변경/삭제될 때까지 보관
* 사용자 및 서버 식별 정보: 관계형 데이터 무결성 및 감사 기록에 필요한 범위에서 보관
* 감사 로그: 운영 및 책임 추적 목적을 위해 보관
* 생성된 브리핑 및 기사 메타데이터: 서비스 기록, 조회, 캐싱 및 안정성을 위해 보관되며, 삭제되지 않는 한 유지될 수 있음

보관 기간은 서비스 변경에 따라 달라질 수 있습니다.

## 7. 데이터 공유

New2Signal은 개인정보를 판매하거나 임대하지 않습니다.

정보는 봇 운영에 필요한 인프라, 데이터베이스, 로깅, AI 서비스 제공자 등 서비스 제공자와 필요한 범위 내에서만 공유될 수 있습니다.
